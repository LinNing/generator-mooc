> Yeoman 生成器，用于生成前端项目/组件文件结构。


## 安装

- **安装 yo**

    ```bash
    $ npm install -g yo
    ```

- **安装 generator**

    ```bash
    $ npm install -g generator-mooc
    ```

- **安装 bower**

    ```bash
    $ npm install -g bower
    ```

## 使用

### 项目目录结构

- **生成**

    ```bash
    $ yo nd


### 组件目录结构

- **生成**

    ```bash
    $ mkdir <name>
    $ cd <name>
    $ yo mooc:module
    ```
